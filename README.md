An end-to-end automation project that utilizes Machine Learning for Sentiment Analysis to enhance customer engagement. The system classifies user comments as positive or negative and triggers automated workflows using n8n.
​Key Features:
​AI Classification: Uses a sentiment analysis model to categorize feedback in real-time.
​Automated Response System: Integrated with n8n to send personalized responses based on the sentiment:
​Negative Feedback: Sends an automated apology and a link to a feedback form to resolve the issue.
​Positive Feedback: Sends an automated thank-you message to build customer loyalty.
​Workflow Integration: Seamlessly connects Google Colab/Python logic with automation tools to streamline business operations.
